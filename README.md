# Saintfield Baptist Church

[saintfieldbaptist.org.uk](https://www.saintfieldbaptist.org.uk)

Website for Saintfield Baptist Church, 51 Crossgar Road, Saintfield.

## Tech Stack

- **Framework:** React 18
- **Build Tool:** Vite
- **Routing:** React Router v6
- **Testing:** Vitest + jsdom
- **CI/CD:** GitHub Actions (build, test, deploy on push to master)

## Getting Started

```bash
npm install
npm start
```

## Scripts

| Command | Description |
|---------|-------------|
| `npm start` | Start dev server |
| `npm run build` | Production build to `dist/` |
| `npm test` | Run tests |
